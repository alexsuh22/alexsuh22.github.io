Java Glossary

Important Terms to Know

Word Definition
Compiler Translates your higher level code into machine code so the computer knows how to run your program
Argument An actual parameter in a method call, or one of the values combined by an operator
Array Data structure that holds a collection of values of the same type stored in contiguous memory locations, each of which can be accessed by an integer index
Math.round(NUMBER); Rounds the number up or down to a whole number
public describes a feature (ex. a method) that can be accessed by all classes
private something that is only accessible by the same class or one of its inner classes
void added to a method where nothing is returned
Constructor Builds object in memory must be public
Accessor Get, which returns the value of the private member variable
Mutator Set, to control changes to a variable
