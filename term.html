Navigation

Command Explanation
ls
  • -a
  • -l
  • -t
 List all files in directory
  • Shows all contents starting with dot (.) [AKA hidden files]
  • Shows all contents in long format
  • -Orders contents by time last modified
[Can be used interchangibly (ls-at)]
pwd Print Working Directory
cd NEWDIRECTORY Change Directory to a new directory
mkdir Make Directory
touch NEWFILE Creates empty file

Command Explanation

mv FILENAME NEWDESTINATION
