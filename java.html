Math Functions

import java.lang.Math;

Function Explanation Example
Math.abs(NUMBER); Shows the abosolute value of the number Math.abs(5-8) = 3
Math.pow(BASE, INDEX); Takes the base and puts it to the power of the index Math.pow(5, 3) = 125
Math.sqrt(NUMBER);
Math.cbrt(NUMBER);		 Finds the square root of the number
Finds the cube root of the number		 Math.pow(5, 3) = 125
Math.round(NUMBER); Rounds the number up or down to a whole number Math.round(132.2) = 132 / Math.round(164.5) = 165
Math.random() * HOWMANY + START; Randomly creates a number from start adding up to howmany Math.random() * 5 + 1 = a number from 1 to 6
